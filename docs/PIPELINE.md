# LLO 解体碎片对地月对接任务风险分析 — Pipeline

**场景**：中国「近地一次环月两次交会」载人登月（He Boyong 2020，国防科大学报）。载人飞船
沿**绕月自由返回轨道**抵达 **100 km 逆行 LDO**，与先期驻留 23 天的着陆器组合体做**第一次
环月交会对接 (1st RVD)**。若此刻 LDO 上目标解体，评估沿自由返回**单次穿越**的载人飞船碰撞风险。

**模型**：场景轨迹用 **CR3BP**（旋转系，DU=389703 km，论文 J2000 月心根数转入）；碎片传播用
**BCR4BP**（加太阳摄动，与原论文一致）。碰撞用**危险区通量模型** + 泊松 `P = 1 − exp(−E)`。

## B. 核心物理 (analysis_ldo.py)

- **危险区 DZ**：以航天器为心、半径 `R_DZ=100 km` 球；`V_DZ=(4/3)πR_DZ³`。
- **航天器截面**：直径 20 m 球壳 → `σ = π·(0.01 km)² ≈ 3.14e-4 km²`（10 m 半径各向同性包络）。
- **期望撞击数**：`E = Σ_i ∫ [碎片 i 在 DZ 内]·v_rel,i·σ/V_DZ dt`（每碎片贡献 e_i）。
- **碰撞概率**：`P = 1 − exp(−E)`（泊松）。
- **缩放**：`Ê = (N_PHYS/N_SUB)·Σ e_i`。
- **⚠️ N_PHYS vs N_SUB（关键区分）**：
  - `N_PHYS` = **真实物理碎片数** = NASA SBM 幂律预测的 >0.1m 碎片数（如 **655**，按航天器质量/能量算）。E ∝ N_PHYS。
  - `N_SUB`(5M) = **蒙特卡洛样本数**，仅为平滑 dv 分布、降方差；最后乘 N_PHYS/N_SUB 缩放。
  - **绝不能把样本数当物理数**（曾误设 N_PHYS=5e6，高估 P 约 5e6/655≈7600×）。
- **为何 5M 样本足够（不需 1e8）**：用的是**通量/密度模型**而非命中法。MC 方差由"进入 100km 危险区的碎片数 n_DZ(~1000s)"决定，相对误差 ≈ 1/√n_DZ ≈ 几 %，**与 E~1e-8 的小量级无关**。100km 危险区即方差缩减核。任务2a 收敛图直接量化此精度。
- **残留率**：碎片距月心 `R_M < d < 0.01 DU` 的占比随 t（排除撞月）。与 N_PHYS 无关。
- **预警时间 Δt**：解体 → 载人飞船近月点(1st RVD) 提前量，物理范围 [0, 23d]（驻留时长）。

## C. 关键发现（新场景 vs 原论文）

1. **残留率 72–74%**（服务器验证）：0.01 DU 阈值 + 撞月冻结碎片排除后 ≈ 论文 70%。
2. **碰撞概率 ~1e-8**（N_PHYS=655 正确缩放后）：单次解体对单次对接为**稀有事件**，与原论文
   1e-9~1e-8 同量级。共面对接几何使其**在相同 N_PHYS 下**高于一般飞掠，但绝对值仍低 →
   真正关注点应转向**累积风险**（多次解体 / 未来交通增长）。
   （注：曾因 N_PHYS=5e6 误算出 ~1e-4，是把样本数当物理数所致。）
3. **Δt=0 P=0（不是上界）**：解体瞬间碎片云仍是一个点，还未扩散到航天器近月穿越点；
   随后碎片绕 LDO 扩散，P 从 0 爬升，**约 Δt≈5d 达峰**，之后衰减并有周期性次峰（脉冲型）。
4. **峰值 Δt≈5d 对所有解体点共同；只有峰值"高度"随相位变**（数据修正，N_PHYS=655 后实测）：
   6 个等相位解体点**峰值 Δt 都≈5d**（1.2–1.8e-7），峰值时刻由**碎片云纵向扩散/近月滞留时间尺度**
   决定（对所有点相同），解体相位只改变峰值高度、不改变时刻。
   （保留: 峰区网格粗(4/5/7d)都吸附到 5d；finer 网格[4,4.5,5,5.5,6]可确认是否有亚天级错位。）
5. **撞月冻结碎片已正确排除**（BCR4BP 冻结判据 d < R_M_LU）。
