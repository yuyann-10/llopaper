# 论文表述修订指南：从「标称自由返回轨道」改为「环月交会对接到达」场景

核心转变：原文是**泛化的"航天器沿自由返回轨道做地月转移"** → 现在是**特定任务：
载人飞船沿绕月自由返回轨道抵达 100km 逆行 LDO，与先期驻留的着陆器组合体做
第一次环月交会对接（He Boyong 2020「近地一次环月两次交会」架构）**。

## 五、需新增的论点（新场景特有，建议加入）

1. **预警时间 Δt 框架**：解体早于对接 Δt 天 → 碰撞概率 P(Δt)。物理范围 [0, 23d]（LOR 驻留时长），
   三个历元节点：lander LOI(23d)/crew TLI(2.6d)/RVD(0d)。**P 峰在 ~5d**。
2. **脉冲/相位结构**：单次穿越只撞到"那一刻在穿越点的碎片"；碎片绕环漂移 → P 随 Δt 呈脉冲型。
   6 个等相位解体点**峰值 Δt 都≈5d**（由碎片云纵向扩散/近月滞留尺度决定），**只峰值高度随相位变**。
3. **空间重叠图景**（图 fig_dist_5d）：5d 时碎片铺满 100km LDO 环，**航天器近月点正好穿过此环** → 风险峰；
   邻域占比曲线（图8）也在 ~5d 达宽峰，两条独立曲线互相印证。
4. **N_PHYS=655 下的稀有事件 + 通量模型**：单次解体对单次对接 P~1e-7 为稀有事件；
   通量/密度模型（非命中法）使 MC 方差由"进危险区碎片数"决定，**5M 样本即达几个 % 精度**（任务2a 收敛图）。
5. **结论转向**：单次解体风险低，但应关注**累积风险**（多次解体 / 未来 LOR 交通增长）。
